- **Population jeune** : 72,6% < 30 ans (18,6 millions selon RGPH3 2024)

## 🎯 Opportunités pour Nexio.work

### Partenariats B2B recommandés
1. **ESTI** - Modèle parfait d'adéquation école-entreprise
2. **ISPM / IT University** - Volume étudiants + réseau entreprises
3. **Work IT Mada / Codeko.tech** - Déjà sur le placement IT
4. **MadaJOB** - Leader recrutement, 150 offres/jour

### Niches non exploitées
- **Certification par IA** : Aucun acteur ne fait de quiz IA automatisés pour la certification
- **Micro-travail local** : Pas de plateforme type "Contra" à Madagascar
- **Edge AI / Offline** : Aucune solution de mentorat IA local (WebGPU) identifiée
- **RAG évolutif** : Aucune plateforme n'enrichit son IA avec les discussions utilisateurs

### Menaces / Concurrence
- **D-CLIC / Asan'Ai** : Financements internationaux massifs, risque de monopole gratuit
- **ESTI / GE-IT** : Formations en alternance, directement absorbées par les entreprises
- **MadaJOB / Work IT Mada** : Déjà positionnés sur le B2B recrutement

### Recommandations stratégiques
1. **Se différencier par l'IA** : Mentorat Gemma 4 local, pas besoin de cloud
2. **Cibler les universités** : ISPM, IT University, HEI pour la base B2C
3. **Partenariats B2B** : Intégrer Nexio.work comme outil de certification pour ESTI, GE-IT
4. **SEO prioritaire** : "Recrutement IT Madagascar", "Certification dev Madagascar", "Simulation entretien IA"
